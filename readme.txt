The easiest way to put a favicon on your Wordpress & Wordpress Mu website

== Description ==

This plugin desinged for Wordpress & Wordpress Mu websites.It is support admin dashboard and web interface.
Also you can translate this plugin.
That's all,have fun :)

== Installation ==


1. Upload `lettoblog-favicon` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your Favicon Options
4. Enter favicon url
5. That's it!
